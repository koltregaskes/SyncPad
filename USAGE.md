# Usage

## First Run

1. Start the app.
2. Create a new note from the sidebar.
3. Type straight into the editor.
4. Wait for autosave to finish.

## Main Actions

- Create notes
- Rename notes by editing the first line or note title
- Search notes from the sidebar
- Delete notes you no longer need
- Keep working locally even before cloud sync exists

## Current Limitations

- Local storage only for now
- No rich formatting
- No multi-device sync yet
