# Setup

## Requirements

- Windows
- Node.js
- npm

## Install

```powershell
npm.cmd install
```

## Run

```powershell
npm.cmd start
```

## Verify

```powershell
npm.cmd run check
npm.cmd run smoke-store
```

## Local Data

By default, SyncPad stores notes in:

`C:\Users\<you>\AppData\Local\MyData\SyncPad\notes.json`

You can override the base directory with:

`MYDATA_DIR`
